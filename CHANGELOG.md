### Added
- Initial repository structure with MIT license

- Statusline scripts for Claude Code, Codex, and OpenCode
- Example configurations for each supported tool
- Comprehensive documentation: installation, customization, troubleshooting
- Basic shell tests for all script families
- GitHub Actions CI workflow with ShellCheck linting and test runner
- GitHub issue templates (bug report, new statusline idea)
- Pull request template with contribution checklist
